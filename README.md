# Kyanite Nuker
One of the **Fastest** discord Server Nukers - coded **FULLY** in **PYTHON**



## Features

- **Nuke**
  - Delete Channels
  - Delete Roles
  - **Create Channels**
    - Spam Channels
    - Spam Webhooks
  - Create Roles
- **YAML Implementation**
  - Token
  - **Channels**
    - Multiple Channel Names
    - Custom Amount
  - **Roles**
    - Multiple Channel Names
    - Custom Amount
  - **Webhooks**
    - Custom Message
    - Custom Names
     
## Installation

```python
pip install discord
pip install requests
pip install pyyaml
```
## Usage/Examples

python main.py
